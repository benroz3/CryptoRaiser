## Deploy

### Locally
1. `cd client`
2. Run `yarn` or `npm i` to install dependencies
3. Run `yarn dev` or `npm run dev` to start the dev server

### With Docker
1. Open Docker Desktop
2. `cd client`
3. Run `docker build -t CryptoRaiser .`
4. Run `docker run -p 3000:3000 CryptoRaiser`
